about sfm

This is a simple app that helps staff at GW Libraries manage one or more feeds of data from social media sites like Twitter. Here you can see the content from those feeds and in some cases link back to the original items on their source sites.

A detailed description of the data reported while enlisting feeds from any social media sites like twitter can be found at Data Dictionary.Here you can see the data which is and can be sent as a report of the tweet for any particular user.
